# Jogo-de-Adivinha-o-em-C

## Sobre o projeto
Jogo simples feito em C onde o jogador precisa adivinhar um número aleatório gerado pelo sistema.

## Como funciona
- O programa gera um número aleatório
- O jogador tenta adivinhar o número
- O sistema dá dicas:
  * "Maior" ou "Menor"
- O jogo continua até o jogador acertar
